# C Project Template

Clone the repo, modify the `Makefile` with the appropriate binary name; that's it.

### Compiling

```
make
```

### Running

```
make run
```

### Tests

```
make test
```

### Valgrind checks

On `Linux`:

```
make valgrind
```

On `macOS` (requires `Docker`):

```
make docker_valgrind
```
